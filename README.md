# jirate

### Add Basic Text Comment to Issue

```sh
jirate add {IssueID} {short content}
```

### Add Comment to Issue via Markdown Editor

```sh
jirate add {IssueID} md
```

### List Comments for Issue By ID

```sh
jirate list {IssueID}
```

### Delete Comment

```sh
jirate delete {IssueID} {CommentID}
```
